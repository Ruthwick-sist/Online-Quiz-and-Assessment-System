# Online-Quiz-and-Assessment-System
This Mini Project Consists of a Java Online Quiz System allowing users to take multiple-choice quizzes, get automatic scores, and view result summaries. Supports question management, simple file storage, and helps learners test their knowledge in an interactive way.
## Features

- Users can attempt quizzes with multiple-choice questions
- Admin can add questions and answers
- Automatic score calculation after quiz completion
- Result summary and performance analysis
- Option to store questions and results using files
- Interactive and simple to use
